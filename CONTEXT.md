# Markdown Editor

A lightweight Mac desktop application for viewing and editing Markdown files, with Typora-level rendering quality and minimal editing complexity.

## Language

**Preview Pane**:
The right-side panel that renders Markdown into styled HTML with full feature support (code highlighting, LaTeX, Mermaid, etc.).
_Avoid_: Viewer, render view

**Source Pane**:
The left-side panel containing a plain textarea for editing raw Markdown source. Can be collapsed to show preview only.
_Avoid_: Editor, code view

**Split Mode**:
The default layout showing Source Pane and Preview Pane side by side.
_Avoid_: Dual pane, two-column

**Preview Mode**:
The layout when Source Pane is collapsed, showing only the rendered Preview Pane.
_Avoid_: Read mode, view mode

**Document**:
A single .md file opened by the user. The app handles one Document at a time.
_Avoid_: File, note, page
