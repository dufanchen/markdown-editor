## 为什么是它

你产出了一堆 Markdown 文档，却发现手头没有一个**简单、免费、好看**的工具来看和改。在线编辑器要联网、要登录；重量级 IDE 杀鸡用牛刀；系统自带的文本编辑器又渲染不了。

这个工具就是来解决这件小事的：双击 `.md` 文件直接打开，左边写、右边实时预览，所见即所得。基于 Tauri 构建，安装包不到 10 MB，启动快、占用低，像一个原生 macOS 应用该有的样子。

## ✨ 功能特性

### 📖 分栏实时预览
左侧写源码、右侧看效果，默认 1:3 黄金比例。拖动中间的分隔线自由调整宽度，双向同步滚动让编辑和预览始终对齐。源码栏还能一键折叠，进入纯预览的沉浸阅读模式。

### 📑 多文档标签页
同时打开多个文档，标签页之间随手切换。未保存的文档会标上 ● 小圆点提醒你。标签支持**拖拽排序**，每个标签**各自记住滚动位置**——切回来还在你离开的地方。

### 🔗 系统文件关联
安装后，`.md`、`.markdown`、`.mdown`、`.mkd`、`.mkdn`、`.txt` 六种文件在访达里双击就能用本应用打开，像处理任何原生文件一样自然。

### 📚 目录导航
右侧自动从标题生成树形目录，点击平滑跳转，滚动时高亮当前所在章节。还会智能跳过代码块里的"伪标题"，不会被 `# 注释` 误导。

### 📁 文件目录
左侧 IDE 风格的树状文件浏览器，打开文件后自动展示所在目录结构。文件夹点击展开 / 折叠，竖线引导层级；非 `.md` 文件自动置灰。支持向上导航、回到当前文件目录，宽度可拖拽调整。从目录中打开新文件不会重置已展开的目录结构。

### 🔍 字体缩放
工具栏一键放大 / 缩小，五档字号（特小 → 特大）同时作用于编辑区、预览区和目录栏，并自动记忆，下次打开保持你的习惯。

### 🌓 日间 / 夜间模式
一键切换明暗主题并记住偏好。暗色模式下代码高亮完整适配，深夜写作不刺眼。

### 🎨 讲究的排版
- **Monochrome (Ivory Ledger)** 极简单色设计，克制而耐看
- **Jost + Lora + JetBrains Mono** 字体搭配，正文、标题、代码各得其所
- **层级缩进**：预览区按标题层级自动缩进，竖线连接同层内容，结构一目了然
- **双向高亮**：在一侧选中文字，另一侧对应位置同步点亮

## 🧩 完整的 Markdown 渲染

支持你需要的一切语法，开箱即用：

| 能力 | 说明 |
| --- | --- |
| 基础语法 | 标题、列表、表格、引用、链接、图片等 |
| 代码高亮 | 基于 highlight.js 的多语言语法着色 |
| 数学公式 | KaTeX 渲染行内与块级公式 |
| 图表 | Mermaid 流程图、时序图、甘特图等 |
| 任务列表 | `- [ ]` / `- [x]` 可视化勾选框 |
| 脚注 | 标准 Markdown 脚注引用 |
| 标题锚点 | 自动为标题生成锚点，便于跳转 |
| 外部链接 | 点击自动用系统默认浏览器打开 |

## 💻 技术栈

- **框架**：[Tauri v2](https://tauri.app/) + [React 19](https://react.dev/) + TypeScript
- **构建**：[Vite](https://vitejs.dev/)
- **渲染**：markdown-it + highlight.js + KaTeX + Mermaid

## 📦 下载安装

前往 [Releases](https://github.com/dufanchen/markdown-editor/releases) 下载最新的 `.dmg` 文件，打开后将应用拖入 `Applications` 文件夹即可。

> 首次打开若提示来自未识别的开发者，在「系统设置 → 隐私与安全性」中点击「仍要打开」即可。

各版本的更新内容见 [CHANGELOG](CHANGELOG.md)。

## 🛠 本地开发

环境要求：Node.js、Rust 工具链（Tauri 依赖）。

```bash
cd app
npm install
npm run tauri dev
```

常用脚本：

```bash
npm run test       # 运行单元测试（Vitest）
npm run lint       # 代码检查（ESLint）
npm run tauri build  # 打包构建
```

构建产物位于 `app/src-tauri/target/release/bundle/`。

## 📁 项目结构

```
app/
├── src/                  # React 前端
│   ├── App.tsx           # 主界面与交互逻辑
│   ├── renderer.ts       # Markdown 渲染与层级缩进
│   ├── useFileManager.ts # 多标签文件管理
│   └── useTheme.ts       # 主题切换
└── src-tauri/            # Tauri 原生层（Rust）
docs/                     # 产品文档（PRD、ADR）
```
